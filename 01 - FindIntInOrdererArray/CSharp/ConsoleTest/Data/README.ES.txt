Los archivos de datos estarán formado por una línea por cada set de prueba. 
Un set de prueba está formado por el array de datos (números enteros separados por comas sin espacios), el valor a buscar, el resultado que debe devolver el programa, separados entre si por un pipe (|)
Puede haber tantas líneas como set de datos se quieran incluir
Ejemplo:
3,6,7,8,12,15|12|4
3,4,5,6,7,8|3|0
3,4,5,6,7,8|1|-1
